Tips for Efficiency

  • Pressing Enter in the Glossary Term input automatically focuses and copies the output
  • Pressing R in the Output box resets and focuses the Glossary Term input
  • Pressing Alt-R anywhere on the page resets and focuses the Glossary Term input without a hard refresh
  • You can Refresh (Control+R), and still preserve the File URL
  • The Glossary Term gets focused on page load. You don't need to use your mouse! Simply refresh!
